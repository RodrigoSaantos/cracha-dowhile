<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/RodrigoSaantos/cracha-dowhile.git

# Go to server folder
$ cd cracha-dowhile

# Install Dependencies
$ yarn
# or
$ npm install

# Run Aplication
$ yarn dev
# or
$ npm run dev

# Access localhost
http://localhost:3000
```
